# FS - Full Stack Exercises

## Overview

This repository contains a collection of exercises and practice projects for learning full-stack web development. It covers front-end technologies like HTML, CSS, and JavaScript, as well as basic backend concepts.

## Folder Structure

* **html-css/**: Basic HTML and CSS exercises.
* **JavaScript/**: JavaScript exercises and projects.
* **pre-course/**: Preparatory exercises for beginners.
